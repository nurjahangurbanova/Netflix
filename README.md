Bu proje, popüler yayın platformu Netflix'in ana sayfa ve giriş yapma (login) ekranlarının modern web teknolojileri kullanılarak yapılmış bir klonudur. Proje, özellikle responsive (duyarlı) tasarım ve Bootstrap bileşenlerinin kullanımına odaklanılarak geliştirilmiştir.

🚀 Özellikler
Tam Responsive Tasarım: Mobil, tablet ve masaüstü cihazlarla tam uyumlu.

Netflix UI/UX: Orijinal tasarıma sadık kalınarak hazırlanan arayüz.

Dinamik Formlar: Bootstrap sınıflarıyla özelleştirilmiş şık giriş formları.

Modern CSS: Linear gradient arka planlar ve yumuşak geçiş efektleri.

🛠 Kullanılan Teknolojiler
HTML5: Semantik yapı ve içerik düzeni.

CSS3: Özel styling, Flexbox ve konumlandırma.

Bootstrap 5: Hızlı ve kolay responsive yapı için kullanıldı.

Google Fonts: Projenin ikonik görünümü için 'Bebas Neue' yazı tipi tercih edildi.
